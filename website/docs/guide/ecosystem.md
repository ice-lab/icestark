# 社区生态

icestark 生态系统帮助快速创建微前端项目。欢迎参与贡献。

## Repos

这里可以找到一些示例仓库，以及官方提供的脚手架。

+ [icestark-vite-react](https://github.com/maoxiaoke/icestark-vite-react) - 使用 [Vite](https://vitejs.dev/) 创建的 React 框架微应用 Demo。[线上预览](https://icestark-vite-react.vercel.app/)。

+ [icestark-vite-vue](https://github.com/maoxiaoke/icestark-vite-vue) - 使用 [Vite](https://vitejs.dev/) 创建的 Vue 框架微应用 Demo。[线上预览](https://icestark-vite-vue.vercel.app/)。

+ [icestark-multi-tabs](https://github.com/maoxiaoke/icestark-multi-tabs) - 基于 icestark 的 **多页签** 示例 Demo。[线上预览](https://icestark-multi-tabs.vercel.app)。

+ [icestark-angular-12](https://github.com/maoxiaoke/icestark-angular-12) - Angular12 框架接入 icestark 的示例。

+ [vite-plugin-index-html](https://www.npmjs.com/package/vite-plugin-index-html) - 一个实现类似于 webpack-html-plugin 的 Vite 插件，可提供 Vite 应用快速接入 icestark 的能力。

+ [build-plugin-icestark](https://github.com/alibaba/ice/tree/master/packages/plugin-icestark) - [icejs](https://ice.work/) 快速接入 icestark 的插件，帮助 icejs 用户一键接入 icestark。[具体使用文档](https://ice.work/docs/guide/advanced/icestark)。
